Arithmetic Operators in JavaScript

Arithmetic operators are used to perform arithmetic operations on numbers. The most common arithmetic operators are: +, -, *, /, and %.

Other arithmetic operators:

  • ++ : Increment
  • -- : Decrement
  • ** : Exponentiation

Example code:


let a = 10;
let b = 3;

let sum = a + b;        // 13
let difference = a - b; // 7
let product = a * b;    // 30
let quotient = a / b;   // 3.333...
let remainder = a % b;  // 1
let power = a ** b;     // 1000
let inc = a++;          // 10 (after a = 11)
let dec = b--;          // 3 (after b = 2)
