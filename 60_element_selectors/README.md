# 🍇 Element Selectors in JavaScript

This project demonstrates different methods to select and manipulate DOM elements using JavaScript.

## 🧪 Selectors Covered
- `getElementById()`
- `getElementsByClassName()`
- `getElementsByTagName()`
- `querySelector()`
- `querySelectorAll()`
