JavaScript Variable Scope

In JavaScript, the scope of a variable determines where that variable is accessible within the code. There are mainly three types of scope:

  • Global Scope: Variables declared outside any function or block. Accessible everywhere.
  • Function Scope: Variables declared inside a function using var. Accessible only inside that function.
  • Block Scope: Variables declared inside a block (like inside {}) using let or const. Accessible only inside that block.
