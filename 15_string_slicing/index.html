String Slicing in JavaScript

  • String slicing is the process of extracting a part (substring) of a string.
  • JavaScript provides the slice() and substring() methods for slicing strings.
  • slice(start, end) extracts from start index up to, but not including, end.
  • Negative indices in slice() count from the end of the string.
  • substring(start, end) is similar, but negative values are treated as 0. 

// Example:
let text = "JavaScript String Methods";
console.log(text.slice(0, 10));      // "JavaScript"
console.log(text.slice(11));         // "String Methods"
console.log(text.slice(-7));         // "Methods"
console.log(text.substring(0, 10));  // "JavaScript"
console.log(text.substring(11));     // "String Methods"
