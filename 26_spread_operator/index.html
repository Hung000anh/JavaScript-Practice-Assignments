Spread Operator in JavaScript

The spread operator ... allows an iterable such as an array or string to be expanded in places where zero or more arguments or elements are expected.

Examples:


let array = [1, 2, 3, 4, 5];
let max = Math.max(...array);  // Spread array into individual numbers
let min = Math.min(...array);

console.log(array.length);
console.log(max);
console.log(min);

let username = "Bro Code";
let letter = [...username];    // Spread string into characters
console.log(letter);
console.log(letter.join("-"));

let fruits = ["apple", "orange", "banana"];
let vegetable = ["carrots", "celery", "potatoes"];
let newFruit = [...fruits, ...vegetable, "eggs", "milks"];  // Combine arrays

console.log(newFruit);
