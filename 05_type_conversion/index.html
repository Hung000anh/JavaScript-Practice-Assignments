Type Conversion in JavaScript is the process of converting a value from one data type to another, for example from a string to a number, a number to a string, or to a boolean.

  • There are two types of conversion: explicit and implicit.
  • Explicit conversion uses functions like Number(), String(), and Boolean().
  • Implicit conversion happens when JavaScript automatically converts data types during operations. 
// Explicit conversion
const num = Number("123");      // 123 (number)
const str = String(456);        // "456" (string)
const bool = Boolean(0);        // false (boolean)

// Implicit conversion
const result = "5" * 2;         // 10 (number)
const concat = "5" + 2;         // "52" (string)
const boolImplicit = !!"hello"; // true (boolean)
