If Statements in JavaScript

  • The if statement is used to execute code if a specified condition is true.
  • You can use else to execute code if the condition is false.
  • else if allows you to check multiple conditions.

// Basic if statement
let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
}

// if...else statement
if (age < 18) {
    console.log("You are a minor.");
} else {
    console.log("You are an adult.");
}

// if...else if...else statement
let score = 85;
if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else {
    console.log("Grade: C or below");
}

The if statement is fundamental for decision making in JavaScript programs.
