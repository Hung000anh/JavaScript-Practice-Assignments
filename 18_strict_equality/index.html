Strict Equality (===) in JavaScript

  • Strict equality (===) checks if two values are equal and of the same type.
  • It does not perform type conversion, unlike the loose equality operator (==).
  • Use !== for strict inequality. 

// Examples:
console.log(5 === 5);       // true
console.log(5 === "5");     // false
console.log(0 === false);   // false
console.log(null === undefined); // false
console.log("abc" === "abc"); // true

// Loose equality for comparison:
console.log(5 == "5");      // true (type conversion happens)
