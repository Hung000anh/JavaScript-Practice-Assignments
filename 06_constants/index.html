Introduction to const in JavaScript

  • const is used to declare constants, whose values cannot be changed after assignment.
  • You must assign a value when declaring a const variable.
  • You cannot reassign a new value to a const variable.
  • If the value is an object or array, you can still modify its properties or elements. 

const PI = 3.14;
const arr = [1, 2, 3];
arr.push(4); // Valid
// arr = [5, 6]; // Error
